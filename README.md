# CleanSheets

CleanSheets é uma ferramenta para manipulação de planilhas Excel, permitindo a seleção e remoção de colunas de forma interativa.

## Como instalar

1. Baixe o executável mais recente [aqui](https://github.com/m1caelvr/CleanSheets/raw/main/dist/CleanSheets.exe).
2. Execute o arquivo baixado para instalar CleanSheets.

## Como Usar

1. Ao executar o programa, selecione o arquivo no qual deseja excluir as colunas.
2. Em seguida, caso alguma planilha .xlsx / .xls tiver sido selecionada, selecione a tabela que deseja modificar.
3. Por fim, digite o número das colunas que deseja manter ou excluir como 1, 3... que será interpretado como as colunas, nesse caso, A, C...
